# AstroidFx
Scalper FX

### Suggested Trade Settings
* Timeframe: 15M or any higher
* Use trending pair

# Input Parameters
### Account Settings 

* StartLot       = 0.01 

* EquityCompound = 0.1

* StartEquity    = 200

* MagicNumber    = 1234


### MACD Parameters

* MACDOpenLevel  = 12

* MACDCloseLevel = 9

* MATrendPeriod  = 26

### EMA parameters 

* EMA_Fast = 8

* EMA_Normal = 12

* EMA_Slow = 21

* EMA_VerySlow = 50
